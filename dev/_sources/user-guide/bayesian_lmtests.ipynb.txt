    "# Bayesian LM Tests for SDM/SDEM Specification\n",
    "\n",
    "This notebook demonstrates the **6 new Bayesian LM test functions** in `bayespecon` for testing spatial model specification in the SDM (Spatial Durbin Model) and SDEM (Spatial Durbin Error Model) directions.\n",
    "\n",
    "## Background\n",
    "\n",
    "### Classical LM Tests (Koley & Bera, 2024)\n",
    "\n",
    "The classical Lagrange Multiplier (LM) framework for spatial models tests:\n",
    "\n",
    "| Test | H₀ | Alternative | df |\n",
    "|------|-----|-------------|-----|\n",
    "| LM-Lag | ρ = 0 | SAR | 1 |\n",
    "| LM-Error | λ = 0 | SEM | 1 |\n",
    "| LM-WX | γ = 0 | SLX | $k_{wx}$ |\n",
    "| LM-SDM (joint) | ρ = 0, γ = 0 | SDM | $1 + k_{wx}$ |\n",
    "| LM-SLX-Error (joint) | λ = 0, γ = 0 | SDEM | $1 + k_{wx}$ |\n",
    "| Robust LM-Lag-SDM | ρ = 0 (robust to γ) | SDM | 1 |\n",
    "| Robust LM-WX | γ = 0 (robust to ρ) | SDM | $k_{wx}$ |\n",
    "| Robust LM-Error-SDEM | λ = 0 (robust to γ) | SDEM | 1 |\n",
    "\n",
    "### Bayesian Robust Chi-Squared Test (Dogan et al., 2021)\n",
    "\n",
    "The Bayesian analogue replaces the classical score with the **posterior mean of the outer product of scores** and uses the **Neyman orthogonal score** for robust tests:\n",
    "\n",
    "$$T_R = (g_\\psi^*)^\\top (V^*)^{-1} g_\\psi^*$$\n",
    "\n",
    "where the Neyman-adjusted score is:\n",
    "\n",
    "$$g_\\psi^* = g_\\psi - J_{\\psi\\phi \\cdot \\sigma} \\, J_{\\phi\\phi \\cdot \\sigma}^{-1} \\, g_\\phi$$\n",
    "\n",
    "This adjustment ensures the test is **robust to local misspecification** in the nuisance parameter φ, following Bera & Yoon (1993) and Dogan et al. (2021, Proposition 3)."
   ]

    "y = X @ beta_true + np.random.normal(scale=1.0, size=n)\n",

    "## 4. Robust Bayesian LM Tests (Neyman Orthogonal Score)\n",
    "\n",
    "These tests use the **Neyman orthogonal score adjustment** from Dogan et al. (2021, Proposition 3) to ensure robustness against local misspecification in the nuisance parameter. This is the key innovation over the classical Bera-Yoon (1993) approach.\n",
    "\n",
    "The adjustment removes the correlation between the test parameter score and the nuisance parameter score:\n",
    "\n",
    "$$g_\\psi^* = g_\\psi - J_{\\psi\\phi \\cdot \\sigma} \\, J_{\\phi\\phi \\cdot \\sigma}^{-1} \\, g_\\phi$$\n",
    "\n",
    "where $J_{\\cdot \\cdot \\cdot \\sigma}$ denotes information matrix blocks partitioned on $\\sigma^2$."
   ]
  },

    "## 7. Decision Framework\n",
    "\n",
    "The Bayesian LM tests provide a principled decision framework for choosing between spatial model specifications:\n",
    "\n",
    "```\n",
    "                    ┌─────────────────────┐\n",
    "                    │  Fit OLS, SAR, SLX   │\n",
    "                    │  (null models)        │\n",
    "                    └──────────┬──────────┘\n",
    "                               │\n",
    "                    ┌──────────▼──────────┐\n",
    "                    │  LM-SDM Joint Test   │──── Significant ────┐\n",
    "                    │  (H₀: ρ=0, γ=0)     │                    │\n",
    "                    └──────────┬──────────┘                    │\n",
    "                               │ Not significant                 │\n",
    "                               ▼                                ▼\n",
    "                    ┌──────────────────┐           ┌────────────────────┐\n",
    "                    │  LM-SLX-Error    │           │  Robust LM-Lag-SDM │\n",
    "                    │  Joint Test      │           │  vs Robust LM-WX   │\n",
    "                    │  (H₀: λ=0, γ=0) │           │                    │\n",
    "                    └────────┬─────────┘           └────────┬───────────┘\n",
    "                             │                              │\n",
    "                    Significant ──► SDEM           Lag wins ──► SAR/SDM\n",
    "                    Not sig ──► OLS               WX wins ──► SLX/SDM\n",
    "```\n",
    "\n",
    "### Key Advantages of the Bayesian Approach\n",
    "\n",
    "1. **Full posterior distribution**: Credible intervals instead of just point estimates\n",
    "2. **Robust to local misspecification**: Neyman orthogonal score ensures valid inference\n",
    "3. **Prior incorporation**: Informative priors can improve test power\n",
    "4. **Model-specific posteriors**: Each test uses the appropriate null model"
