    "## Panel Model Equations\n",
    "\n",
    "Let observations be stacked by time, then unit (panel_g convention).\n",
    "\n",
    "Fixed-effects / pooled classes:\n",
    "- OLS panel: $y_{it} = x_{it}'\\beta + a_i + \\tau_t + \\epsilon_{it}$\n",
    "- SAR panel: $y_{it} = \\rho (Wy)_{it} + x_{it}'\\beta + a_i + \\tau_t + \\epsilon_{it}$\n",
    "- SEM panel: $y_{it} = x_{it}'\\beta + a_i + \\tau_t + u_{it},\\; u_{it}=\\lambda (Wu)_{it}+\\epsilon_{it}$\n",
    "- SDM panel: $y_{it} = \\rho (Wy)_{it} + x_{it}'\\beta + (Wx)_{it}'\\theta + a_i + \\tau_t + \\epsilon_{it}$\n",
    "- SDEM panel: $y_{it} = x_{it}'\\beta + (Wx)_{it}'\\theta + a_i + \\tau_t + u_{it},\\; u_{it}=\\lambda (Wu)_{it}+\\epsilon_{it}$\n",
    "\n",
    "Random-effects classes:\n",
    "- OLS random effects: $y_{it} = x_{it}'\\beta + \\alpha_i + \\epsilon_{it},\\; \\alpha_i \\sim N(0, \\sigma_\\alpha^2)$\n",
    "- SAR random effects: $y_{it} = \\rho (Wy)_{it} + x_{it}'\\beta + \\alpha_i + \\epsilon_{it},\\; \\alpha_i \\sim N(0, \\sigma_\\alpha^2)$\n",
    "- SEM random effects: $y_{it} = x_{it}'\\beta + \\alpha_i + u_{it},\\; u_{it}=\\lambda (Wu)_{it}+\\epsilon_{it},\\; \\alpha_i \\sim N(0, \\sigma_\\alpha^2)$\n",
    "\n",
    "The `model` argument selects the pooled or fixed-effects specification for the FE classes:\n",
    "- `0` pooled\n",
    "- `1` unit FE\n",
    "- `2` time FE\n",
    "- `3` two-way FE\n",
    "\n",
    "The RE classes are hierarchical models with unit-level random intercepts, so they internally use the pooled design and estimate `sigma_alpha` directly."
   ]

    "def fit_panel_model(\n",
    "    model_cls, formula, data, W, model=3, draws=300, tune=300, chains=2, seed=42\n",
    "):\n",
    "    \"\"\"Fit a panel model and return (model, summary, effects_df).\n",
    "\n",
    "    Uses minimal MCMC settings for pedagogical demonstration.\n",
    "    Increase draws/tune for real analyses.\n",
    "    \"\"\"\n",
    "    m = model_cls(\n",
    "        formula=formula,\n",
    "        data=data,\n",
    "        W=W,\n",
    "        unit_col=\"unit\",\n",
    "        time_col=\"time\",\n",
    "        model=model,\n",
    "        logdet_method=\"eigenvalue\",\n",
    "    )\n",
    "    m.fit(\n",
    "        draws=draws,\n",
    "        tune=tune,\n",
    "        chains=chains,\n",
    "        target_accept=0.9,\n",
    "        random_seed=seed,\n",
    "        progressbar=False,\n",
    "        idata_kwargs={\"log_likelihood\": True},\n",
    "    )\n",
    "    summary = m.summary(round_to=3)\n",
    "    effects_df = pd.DataFrame(m.spatial_effects())\n",
    "    return m, summary, effects_df\n",
    "\n",
    "\n",
    "def diagnostics_table(idata, var_names):\n",
    "    \"\"\"Show key MCMC diagnostics for the given parameters.\"\"\"\n",

    "    return az.summary(idata, var_names=var_names, round_to=3)[cols]\n",

    "\n",

    "    plt.suptitle(title, y=1.02)\n",

    "### Interpreting the Panel LM Decision Tree\n",
    "\n",
    "The panel LM tests follow the same decision logic as the classical Anselin/Koley-Bera approach, but with Bayesian p-values:\n",
    "\n",
    "| Scenario | Significant? | Interpretation |\n",
    "|---|---|---|\n",
    "| LM-Lag ✓, LM-Error ✗ | Lag only | Choose SAR panel |\n",
    "| LM-Lag ✗, LM-Error ✓ | Error only | Choose SEM panel |\n",
    "| Both significant → check robust tests | | |\n",
    "| Robust LM-Lag ✓, Robust LM-Error ✗ | Lag dominates | Choose SAR panel |\n",
    "| Robust LM-Lag ✗, Robust LM-Error ✓ | Error dominates | Choose SEM panel |\n",
    "| Both robust significant | Both effects | Consider SDM or SDEM |\n",
    "| Joint LM-SDM ✓ | SDM preferred over OLS | Fit SDM panel |\n",
    "| Joint LM-SLX-Error ✓ | SDEM preferred over OLS | Fit SDEM panel |\n",
    "| LM-WX ✓ (from SAR) | WX effects present | Extend SAR → SDM |\n",
    "| Robust LM-Lag-SDM ✓ (from SLX) | Lag needed beyond WX | Extend SLX → SDM |\n",
    "| Robust LM-WX ✓ (from SAR) | WX needed beyond lag | Extend SAR → SDM |\n",
    "| Robust LM-Error-SDEM ✓ (from SLX) | Error needed beyond WX | Extend SLX → SDEM |\n",
    "\n",
    "A Bayesian p-value below 0.05 (or a threshold you choose) indicates evidence against the null hypothesis. The `credible_interval` field shows the posterior uncertainty in the LM statistic itself."
   ]
